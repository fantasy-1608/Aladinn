# Hướng dẫn Cài đặt Phiên bản Pilot

Quá trình này dành cho các bác sĩ tham gia nhóm Pilot.

## Chuẩn bị

1. Tải về file ZIP mới nhất từ IT Department hoặc GitHub Releases (Ví dụ: `Aladinn-v1.2.4.zip`).
2. Giải nén file ZIP vào một thư mục riêng biệt trên máy tính (VD: `D:\Aladinn`).

## Cài đặt trên Chrome/Edge

1. Mở trình duyệt Chrome hoặc Edge.
2. Truy cập vào đường dẫn `chrome://extensions/` (hoặc `edge://extensions/`).
3. Bật chế độ **Developer mode** (Chế độ nhà phát triển) ở góc trên bên phải.
4. Bấm nút **Load unpacked** (Tải tiện ích đã giải nén).
5. Chọn thư mục `D:\Aladinn` vừa giải nén ở trên.
6. Tiện ích "Aladinn - VNPT HIS AI Assistant" sẽ xuất hiện trong danh sách.

## Kiểm tra sau cài đặt

- Truy cập VNPT HIS.
- Một biểu tượng 🧞 sẽ hiển thị bên góc phải màn hình Chrome.
- Nếu được cung cấp PIN hoặc API Key nội bộ, hãy vào **Settings** để cấu hình.
